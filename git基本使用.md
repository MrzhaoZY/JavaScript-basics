1. mkdir 文件名   => 创建文件夹
2. mkdir test => 创建一个test的文件夹  => cd test 进入test
3. git init 初始化仓库
4. touch 文件名 => 添加文件   touch hello.md
5. git status
6. git add 文件名 将文件提交到本地仓库
7. git commit -m '描述'   把文件从暂存区提交到仓库
8. rm 文件名 删除文件
9. git rm 文件名 从git中删除文件



// 提交到远程仓库

git 克隆操作  目的： 将远程仓库复制到本地

git clone 仓库地址

git push 将本地仓库同步到git远程仓库中
